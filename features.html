How we work

A focused approach to secure communication

A focused approach to secure communication

01
Discover

Locbit scans for nearby BLE-enabled devices automatically and lists them as available contacts.

02
Connect

A secure BLE handshake is established between devices, verified via key exchange, with no server involved.

03
Encrypt

Messages are encrypted on-device using AES-256 before transmission. Keys never leave your device.

04
Transmit

Encrypted data is sent peer-to-peer over BLE. Received and decrypted only on the target device.

FAQ

You ask. We answer

You ask. We answer

(01)
What is Locbit?

Locbit is a secure, offline-first messaging app that works entirely via Bluetooth Low Energy — no internet, no servers, no surveillance.

(02)
Who is Locbit for?

Locbit is built for anyone who needs privacy — activists, journalists, disaster responders, remote teams, security-conscious individuals, and developers building secure comms.

(03)
Does Locbit require an internet connection?

No. Locbit works entirely offline using BLE (Bluetooth Low Energy). It runs without Wi-Fi, mobile data, or any central server.

(04)
Is it secure?

Yes. All messages are encrypted with AES-256-GCM using keys exchanged via ECDH. No data ever leaves your device to a server.

(05)
Where can I get Locbit?

Locbit is available as an open-source Android APK. Visit our GitHub repository to download the latest release and start messaging securely.
